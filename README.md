# ☕ Café Atelier — Single Page Website

## 🧾 Descripción

**Café Atelier** es una **Single Page Website (SPA)** desarrollada para representar la identidad de una cafetería moderna, combinando diseño elegante, navegación fluida y enfoque en la experiencia del usuario.

El sitio presenta todo el contenido en una única página, organizada en secciones estratégicas: inicio, presentación de marca, menú y contacto. La navegación se realiza mediante scroll y anclas internas, evitando recargas y generando una experiencia más dinámica e intuitiva.

---

## 🎯 Objetivo

El objetivo del proyecto fue diseñar y desarrollar una **landing page atractiva y funcional**, orientada a:

* Transmitir la identidad visual de la marca
* Destacar productos principales (menú)
* Facilitar el acceso a información clave (ubicación, horarios, contacto)
* Mejorar la experiencia del usuario mediante navegación simple

Este tipo de solución es ideal para pequeños negocios que buscan una **presencia digital moderna, clara y efectiva**.

---

## 💻 Tecnologías utilizadas

* **HTML5**
  Estructuración semántica del contenido y organización de las secciones del sitio.

* **CSS3**
  Estilización completa del proyecto, incluyendo tipografía, colores, layout y responsive design.

* **Flexbox & CSS Grid**
  Utilizados para construir layouts flexibles y adaptables a distintas resoluciones.

* **Google Fonts**
  Implementación de tipografías elegantes (*Playfair Display* y *Inter*) para reforzar la identidad visual.

* **Bootstrap Icons**
  Integración de iconografía moderna para redes sociales.

---

## ⚡ Características principales

* Diseño **Single Page (SPA)**
* Navegación con anclas internas (`#inicio`, `#menu`, etc.)
* Navbar fijo con efecto **glassmorphism (blur + transparencia)**
* Diseño **responsive** adaptado a:

  * Desktop
  * Tablet
  * Mobile
* Sección de menú con **cards visuales de productos**
* Integración de redes sociales
* Identidad visual consistente (colores, tipografía y estilo)

---

## 📱 Responsive Design

El proyecto incluye un sistema de adaptación mediante **media queries**, optimizando la experiencia en diferentes dispositivos:

* **Desktop:** layout completo con grid de 4 columnas
* **Tablet:** reducción a 2 columnas
* **Mobile:** diseño en columna única con navegación adaptada

## ⚡ ¿Por qué una Single Page?

Se eligió una arquitectura **Single Page** por sus ventajas:

* Navegación rápida sin recargas
* Mejor experiencia de usuario (UX)
* Diseño moderno y minimalista
* Mayor enfoque en conversión (contacto / pedidos)
* Ideal para negocios gastronómicos y portfolios
